State Codes
===========

List of the state codes representing the names of states and federal territories in Malaysia.

[Codes](/codes/)

Overview
========

State codes are used to supply precise state information when providing address information as part of the document submission. State codes used in documents are following [ISO-3166](https://www.iso.org/obp/ui/#iso:code:3166:MY).
