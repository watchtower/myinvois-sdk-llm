Overview
========

Tax types are used when entering tax information as part of the invoice (where applicable).

List
====

Taxable Types
-------------

| Code | Description |
| --- | --- |
| 01 | Sales Tax |
| 02 | Service Tax |
| 03 | Tourism Tax |
| 04 | High-Value Goods Tax |
| 05 | Sales Tax on Low Value Goods |
| 06 | Not Applicable |
| E | Tax exemption (where applicable) |
