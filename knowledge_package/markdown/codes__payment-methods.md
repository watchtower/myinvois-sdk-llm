Payment Modes
=============

Payment Mode refers to the chosen mechanism through which funds are transferred from buyer to supplier.

[Codes](/codes/)

Overview
========

Payment Mode that may be selected at the time of document submission.

List
====

| Code | Description |
| --- | --- |
| 01 | Cash |
| 02 | Cheque |
| 03 | Bank Transfer |
| 04 | Credit Card |
| 05 | Debit Card |
| 06 | e-Wallet / Digital Wallet |
| 07 | Digital Bank |
| 08 | Others |

=========================
