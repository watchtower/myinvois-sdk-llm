e-Invoice Types
===============

List of e-Invoice types allowed as part of the document submission.

[Codes](/codes/)

Overview
========

e-Invoice types are used to identify the document type.

List
====

| Code | Description |
| --- | --- |
| 01 | Invoice |
| 02 | Credit Note |
| 03 | Debit Note |
| 04 | Refund Note |
| 11 | Self-billed Invoice |
| 12 | Self-billed Credit Note |
| 13 | Self-billed Debit Note |
| 14 | Self-billed Refund Note |

File Download
=============

[Download JSON file](/files/EInvoiceTypes.json) containing codes of all the e-invoice types.

Additional Considerations
=========================

Note that when submitting the documents, taxpayers should be using the code values.
